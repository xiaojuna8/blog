全局变量

2018/1/30 posted in  C++

C/C++多文件使用全局变量

  1. 头文件声明extern
  2. 实现文件初始化
  3. 其它调用位置包含声明头文件

若仅在 头文件声明初始化，编译器报链接错误

若声明不是extern也会有链接错误

example

//fun.h
extern int globalInt;
extern int globalArray[3];

//fun.cpp
int globalInt = 123;
int globalArray[3] = {1,2,3};

//main.cpp
#include "fun.h"
int main()
{
  //access globalInt
  //access globalArray;
  return 0;
}

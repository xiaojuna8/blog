GCC编译静态库

代码编写

需要导出的库用pragma GCC visibility push(default)包围

/* The classes below are exported */
#pragma GCC visibility push(default)

class StaticLibTest
{
    public:
    void HelloWorld(const char *);
};

#pragma GCC visibility pop

编译

首先编译为object文件，然后用ar工具编译为静态库.a文件

g++ -c -o out.o StaticLibTest.cpp
ar rcs libout.a out.o

链接

若静态库与原文件相同目录，则直接引用

g++ -o main main.cpp libout.a

混合链接

成功的，控制混合类型的链接(静态库和动态库)

g++ -o main main.cpp -L. -lout -ldy
  1. 指定库目录-L，其中-L.表示当前目录
  2. 指定库-l,其中-lout表示链接库libout首先尝试动态库然后在尝试静态库

在stackoverflow上找到的方法，但是没有成功

g++ -Wl,-Bstatic -lz -lfoo -Wl,-Bdynamic -lbar -Wl,--as-needed

Will link zlib and libfoo as static, and libbar as dynamic . --as-needed will drop any unused dynamic library.

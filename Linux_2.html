cenos 安装 gcc/clang

yum

yum list

yum install xxx

gcc

yum install gcc

yum install gcc-c++

yum install gdb

clang

yum install clang

yum install lldb

安装高版本

上面命令都是安装稳定版编译器，要安装高版本gcc/clang需要特殊设置

By enabling the SCL repository you can install Clang 4 on CentOS 7 using the llvm-toolset-7package.

$ sudo yum install centos-release-scl
$ yum install llvm-toolset-7

yum list | grep gcc

yum install devtoolset-7-gcc
scl enable devtoolset-7 bash

yum install llvm-toolset-7-clang
scl enable llvm-toolset-7 bash

gcc -v
clang -v

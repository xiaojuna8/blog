磁盘传输

2018/1/30 posted in  Computer

2017.11.28

  • SATA
  • NVMe
  • 磁盘

SATA

SATA是Serial ATA的意思，是连结物理存储设备的接口。在早些时用PATA接口。

Generally, the actual SATA signalling is half-duplex, meaning that it can only read or write data at any one time.

SATA是半双工模式，意味着同一时刻，只能读或者写。

Name Raw Rate Gbit/s Max Data Rate
SATA1.0 1.5 150MB/s
SATA2.0 3.0 300MB/s
SATA3.0 6.0 600MB/s
SATA3.2 16.0 1.97GB/s

NVMe

依托于PCIe总线，NVMe设备可适用于各种支持PCIe总线的物理插槽上，接口速度可达3GB以上。

例如三星的SSD数据，读可到3GB／s，写可到2GB／s

其中MLC成本高于TLC，性能也高。

磁盘

普通机械硬盘，读写速度一般200MB／s内。

使用SATA3.0的SSD最大读写500MB／s左右。

NVMe SSD读3GB／s写2GB／s
